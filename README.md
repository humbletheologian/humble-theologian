# Humble Theologian GitHub Pages Starter

This is a Jekyll/GitHub Pages starter site for **Humble Theologian**.

Tagline: **Learning, loving and living the faith.**

Design direction:

- Nook-inspired
- creation palette: earth, sky, starlight and deep space
- content first
- subtle menu and headings
- long-form readability
- designed for Bible college students and serious theologically minded Christians

See `GITHUB-PAGES-SETUP.md` for deployment steps and `CONTENT-GUIDE.md` for the writing standard.


## Local preview

Do not open the root `index.html` directly for preview. It is a Jekyll source file. Open `static-preview/index.html` instead to see the styled site locally.
