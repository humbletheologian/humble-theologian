## Spurgeon

The Spurgeon devotional PDFs from the old site have been converted into web pages in the `_spurgeon` collection.

- Converted Spurgeon pages: 15
- Public listing: Resources page
- GitHub Pages output path: `/spurgeon/[devotional-slug]/`

The site attributes these to Charles H. Spurgeon. Original Spurgeon writings are public domain due to his death in 1892. If any uploaded PDF contains modern editorial material, modernised text or publisher-added content, that should be reviewed before public launch.

## Storage

The site now uses text pages instead of uploading hundreds of PDFs. This keeps the package small and avoids the storage issue that would have arisen on a free WordPress plan.

## Launch status

The package is suitable for GitHub upload and test deployment. Before pointing `humbletheologian.com` to it, review the content for theological accuracy, permissions and any personal details you do not want public.
