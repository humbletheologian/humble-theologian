# Local Preview Instructions

The root files are Jekyll/GitHub Pages source files, so opening `index.html` directly from the root will show unprocessed template code.

For a normal browser preview, open:

`static-preview/index.html`

This folder is a rendered static preview of the site with CSS, colours, navigation and article pages.

When uploading to GitHub Pages, upload the whole starter package except you do not need to upload `static-preview` if you do not want to. GitHub Pages will build the root Jekyll files automatically.
